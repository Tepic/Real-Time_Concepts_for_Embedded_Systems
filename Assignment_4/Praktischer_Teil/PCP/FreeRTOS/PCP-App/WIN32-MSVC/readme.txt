** c. and .h file Explanations **
1. workerTask.h 
   - Data Structure that stores task information needed for the task dispatcher and ICPP
2. gll.h
   - Generic Linked List Data Structure
3. bool_t.h
   - Typedefs for boolean data type
4. semaphore.h
   - Wrapper for FreeRTOS' semaphore
   - Enables both Priority Inheritance and Immediate Priority Ceiling Protocol   
5. print.h
   - Used to write output to the console
6. config.h
   - Enable/Disable Debug and Scheduler Running mode
   - With Debug Mode Enabled additional output will be logged onto the console
